# 二面介绍话术

MealMind 是一个本地饮食决策 Skill。

第一版不接 Apple Watch 或 HealthKit，而是先做一个更实际的 MVP。

用户提前填写个人信息表，包括固定住宿地址、预算、身体信息、饮食偏好和目标。系统根据固定住址调用授权地图 Provider 查询附近饭店、便利店、超市、饮品店和咖啡店的信息，再结合用户每天输入的心情、饥饿程度、工作强度、是否运动和今日目标，生成饮食推荐。

这个系统不会只说“吃健康一点”，而是会输出具体的餐食决策，包括吃什么、喝什么、去哪里买、大概多少钱、为什么推荐、今天不建议吃什么，以及点餐关键词。

技术上，项目拆成了用户画像、基于位置的附近店铺搜索、POI 餐饮字段解析、候选餐食生成、餐食评分、安全检查和结构化 JSON 输出几个模块。

MVP 使用授权地图 POI 验证推荐逻辑，不用示例菜单伪装真实店铺。后续可以接入 OCR 菜单识别、用户授权上传的价格截图、公开优惠信息，以及 Apple Health 数据。
