# Codex Automation 计划

运行频率：

```txt
0 9,17 * * *
```

自动化任务内容：

```txt
运行 MealMind 饮食推荐流程。

使用固定用户画像、授权地图 POI 情报和高德实时天气。
如果 today_status.json 存在，就使用里面的今日状态。
如果今日状态缺失，就提醒用户提供：

- 心情
- 饥饿程度
- 工作/学习强度
- 今天是否运动
- 今天目标
- 当前餐次

生成当前餐次的饮食推荐。

如果没有授权 POI 情报，不要使用示例菜单补结果，直接说明缺少实时店铺数据。

不要自动点餐。
不要自动付款。

只输出：

- 推荐吃什么
- 推荐喝什么
- 去哪里买
- 预计价格
- 推荐理由
- 点餐关键词
- 今天不推荐项
```
